# Jef Portfolio

A static portfolio redesign inspired by blurred poster design, grain textures, frosted glass, bold typography, and editorial layouts.

Open `index.html` in a browser to preview it. The homepage keeps Climate Change as-is, trims the project selection, makes Dynamic Machine more compact, and presents Hubo Limburg United as a school assignment with a slider. WPL2 now has its own case-study page at `wpl2.html` with selected local visuals from the WPL2 folders.
